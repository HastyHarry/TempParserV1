Extraction Rules

Each rule can work in two modes: regex (from full OCR text) or template zone (fixed x/y/w/h area in % of the page). For strict forms, use template zones for higher reliability.

Pattern Examples

Invoice No: Invoice\s*No[:\s]+(\S+)
Date: Date[:\s]+(\d{2}[\/\-]\d{2}[\/\-]\d{4})
Total: Total[:\s]+\$?([\d,\.]+)
Name: Name[:\s]+([A-Za-z\s]+)

Notion Integration

How to get these:
1. notion.so → Settings → Integrations → New integration
2. Copy the Internal Integration Token
3. Open your database → Share → Invite your integration
4. Copy the database ID from the URL:
notion.so/…/DATABASE_ID?v=…

Notion CORS Proxy

Notion API blocks direct browser requests. Enter a CORS proxy URL, or leave default to use a public one (not recommended for production — host your own).
